## Get Started

1. Clone this repository.
2. Check out the `streak` branch.
3. Run the development server:
    
    ```bash
    npm run dev
    ```
    
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
5. Filter to your heart's content.
